## 🔧 API 接口文档

### 用户管理接口

#### 1. 获取所有用户
- **URL**: `GET /api/users`
- **响应**: `ResponseResult<List<User>>`

#### 2. 获取用户详情
- **URL**: `GET /api/users/{userId}`
- **响应**: `ResponseResult<User>`

#### 3. 创建用户
- **URL**: `POST /api/users`
- **请求体**: `User`
- **响应**: `ResponseResult<User>`

#### 4. 更新用户
- **URL**: `PUT /api/users/{userId}`
- **请求体**: `User`
- **响应**: `ResponseResult<User>`

#### 5. 删除用户
- **URL**: `DELETE /api/users/{userId}`
- **响应**: `ResponseResult<Void>`
