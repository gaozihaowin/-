# 我的课程管理

## 功能名称
**学员课程参与进度与状态管理功能**

## 业务逻辑
该功能在 App 端“我的”页面或专门的“我的课程”模块展示：
1. **课程概览**：展示学员已报名的所有营期，包括进行中和已结束的。
2. **学习进度追踪**：展示已学天数、待打卡任务、作业提交情况等。
3. **活跃课程展示**：优先展示用户最近参与或点击过的课程。

## 技术实现

### Controller 层
在 `CourseListController` 和 `AppCourseController` 中定义了获取课程列表和详情的接口：

```java
    /**
     * 获取全部正在开课的营期列表（分页）
     * 接口路径：GET /courses/list
     */
    @GetMapping("/list")
    public Result<PageInfo<Course>> getCourseList(
            @RequestParam(defaultValue = "1") Integer pageNum,
            @RequestParam(defaultValue = "10") Integer pageSize) {
        PageInfo<Course> pageInfo = courseListService.getCourseList(pageNum, pageSize);
        return Result.success(pageInfo);
    }

    /**
     * 获取指定营期的课程安排目录
     * GET /courses/{campId}/schedule
     */
    @GetMapping("/{campId}/schedule")
   public Result<List<CampScheduleDTO>> getCourseSchedule(@PathVariable Integer campId) {
        List<CampScheduleDTO> scheduleList = courseService.getCourseSchedule(campId);
        return Result.success(scheduleList);
    }
```

### 业务逻辑实现 (Today Course)
在 `AppCourseController` 中获取用户当前的课程学习进度：

```java
    /**
     * 获取指定营期的今日课程
     * GET /courses/{campId}/today
     */
    @GetMapping("/{campId}/today")
   public Result<TodayCourseDTO> getTodayCourse(@PathVariable Integer campId, HttpServletRequest request) {
        Long currentUserId = (Long) request.getAttribute("userId");
        if (currentUserId == null) {
            throw new RuntimeException("用户未登录或 Token 失效，无法访问课程");
        }
        TodayCourseDTO todayCourse = courseService.getTodayCourse(campId, currentUserId);
        return Result.success(todayCourse);
    }
```

### 数据交互 (Enrollment)
通过 `CampServiceImpl` 处理用户报名逻辑：

```java
    @Override
    @Transactional(rollbackFor = Exception.class)
    public void enrollCamp(Long userId, Integer campId) {
        // ... 校验营期存在且可报名 ...
        Integer count = campEnrollmentMapper.countByUserIdAndCampId(userId, campId);
        if (count != null && count > 0) {
            throw new IllegalArgumentException("您已报名过该营期");
        }
        campEnrollmentMapper.insertEnrollment(userId, campId);
        campMapper.incrementEnrollCount(campId);
    }
```

## 接口定义
- **获取我的课程列表**：`GET /courses/list`
- **获取课程目录**：`GET /courses/{campId}/schedule`
- **获取今日任务详情**：`GET /courses/{campId}/today`
- **获取课程数据看板**：`GET /courses/{campId}/data`

## 实现细节
- **分页加载**：使用 `PageHelper` 插件对课程列表进行分页查询，提升大数据量下的响应速度。
- **状态感知**：`getTodayCourse` 接口会根据当前用户的 `userId` 实时计算任务完成状态（已点击/未完成）。
- **事务一致性**：报名操作（`enrollCamp`）通过 `@Transactional` 保证了向 `t_camp_enrollment` 插入记录与 `t_camp` 计数增加的原子性。
- **安全性**：所有用户相关的课程查询均需通过拦截器校验 `Authorization` Header，并在 Controller 中显式获取 `userId`。
