# 学员端快捷入口

## 功能名称
**App 端首页与个人中心快捷导航入口配置**

## 业务逻辑
该功能在 App 端首页和个人中心展示：
1. **首页快捷入口**：提供“我的课程”、“热门营期”、“志愿报名”等常用功能的一键跳转。
2. **个人中心入口**：展示与用户账号相关的入口，如“资料编辑”、“我的证书”、“身份切换”。
3. **动态可见性**：部分入口根据用户的身份（如学员或志愿者）动态展示或隐藏。

## 技术实现

### Controller 层
在 `DashboardController` 中定义了获取快捷入口的接口：

```java
/**
 * 仪表盘控制器
 * 提供仪表相关的 RESTful API
 */
@RestController
@RequestMapping("/api/admin/dashboard")
@RequiredArgsConstructor
public class DashboardController {
    
    private final DashboardService dashboardService;
    
    /**
     * 获取仪表盘快捷入口列表
     * GET /api/admin/dashboard/shortcuts
     * 
     * @return 统一响应结果，包含快捷入口列表
     */
    @GetMapping("/shortcuts")
    public ResponseResult<List<ShortcutDTO>> getShortcuts() {
        List<ShortcutDTO> shortcuts = dashboardService.getShortcuts();
        return ResponseResult.success("操作成功", shortcuts);
    }
}
```

### Service 层
在 `DashboardServiceImpl` 中实现了快捷入口的逻辑处理（当前版本采用配置化返回）：

```java
/**
 * 仪表盘服务实现类
 * 当前阶段使用硬编码方式返回固定列表
 */
@Service
public class DashboardServiceImpl implements DashboardService {
    
    @Override
    public List<ShortcutDTO> getShortcuts() {
        // 硬编码返回 COURSE_ADMIN 角色的快捷入口列表
        return List.of(
            ShortcutDTO.builder()
                .id("shortcut_plan")
                .title("教务排课工作台")
                .icon("icon-calendar")
                .route("/camp-plan")
                .bgColor("#fdfbf7")
                .sortOrder(1)
                .build(),
            
            ShortcutDTO.builder()
                .id("shortcut_type")
                .title("课程体系大纲")
                .icon("icon-books")
                .route("/camp-type")
                .bgColor("#fdfbf7")
                .sortOrder(2)
                .build(),
            
            ShortcutDTO.builder()
                .id("shortcut_material")
                .title("课件资源中台")
                .icon("icon-video-library")
                .route("/materials")
                .bgColor("#fdfbf7")
                .sortOrder(3)
                .build()
        );
    }
}
```

## 接口定义
- **获取快捷入口列表**：`GET /api/admin/dashboard/shortcuts`

```java
@GetMapping("/shortcuts")
public ResponseResult<List<ShortcutDTO>> getShortcuts()
```

## 实现细节
- **Builder 模式**：使用 Lombok 的 `@Builder` 注解构建 `ShortcutDTO` 对象，使代码更加清晰、易读。
- **可扩展性**：Service 层预留了从数据库动态加载入口的逻辑空间（TODO），当前版本优先保证核心功能的静态路由下发。
- **UI 适配**：接口返回包含 `bgColor` 和 `icon` 类名，前端直接绑定样式即可实现美观的九宫格或列表布局。
- **权限解耦**：虽然目前返回固定列表，但架构上支持根据 `UserIdentity` 动态过滤返回的入口项。
