# 项目功能文档总目录

本目录涵盖了 Java 后端项目的核心业务逻辑、技术实现细节及接口定义。文档按功能模块进行划分，方便开发人员快速了解系统架构与实现。

## 1. 认证与用户模块
- [登录与身份认证](认证与用户模块/登录与身份认证.md) —— 涵盖 App 端（账号密码、微信）及 PC 管理端登录，以及多端身份切换逻辑。
- [用户信息管理](认证与用户模块/用户信息管理.md) —— 用户个人资料查询、完整度检查及信息全量更新。

## 2. 教务与排课模块
- [营期生命周期管理](教务与排课模块/营期生命周期管理.md) —— 营期的新增、编辑、状态管理及基础列表查询。
- [营期排课算法与计划](教务与排课模块/营期排课算法与计划.md) —— 一键生成日历框架、单日课表保存与时间轴展示。
- [班级与报名管理](教务与排课模块/班级与报名管理.md) —— 营期报名逻辑及班级关联管理。

## 3. 课程管理模块
- [课程体系与分类](课程管理模块/课程体系与分类.md) —— 课程体系的层级管理与分类查询。
- [热门课程与推荐](课程管理模块/热门课程与推荐.md) —— 基于时间的动态热门课程推荐算法实现。
- [我的课程管理](课程管理模块/我的课程管理.md) —— 学员已参与课程的进度查询与状态追踪。

## 4. 仪表盘与数据看板
- [管理员数据看板](仪表盘与数据看板/管理员数据看板.md) —— 后台首页最近活跃课程及核心指标统计。
- [学员端快捷入口](仪表盘与数据看板/学员端快捷入口.md) —— App 端个人中心及首页的快捷入口配置。

## 5. 作业与任务模块
- [任务完成与记录](作业与任务模块/任务完成与记录.md) —— 学员每日任务的点击、完成状态记录。
- [作业评选与管理](作业与任务模块/作业评选与管理.md) —— 志愿者对优秀作业的标记、取消标记及详情查看。
- [统计分析](作业与任务模块/统计分析.md) —— 班级/组维度的作业提交率、状态名单统计。

## 6. 志愿服务模块
- [权限范围计算](志愿服务模块/权限范围计算.md) —— 志愿者管理范围（班级/大组/小组）的动态计算。
- [服务历史统计](志愿服务模块/服务历史统计.md) —— 志愿者服务时长、统计指标及历史记录查询。

---
*注：所有 API 接口均遵循统一的 ResponseResult 结构。*
