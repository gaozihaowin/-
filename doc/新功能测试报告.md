## 已知问题与注意事项

### ⚠️ 数据库约束检查

请确认数据库中已添加手机号唯一约束：
```sql
ALTER TABLE t_user ADD UNIQUE KEY uk_phone (phone);
```

如果没有添加，手机号重复检测将不会生效。

### ⚠️ 上传目录权限

确保 `C:/camp_system/uploads/` 目录存在且有读写权限：
```bash
# Windows 手动创建目录
mkdir C:\camp_system\uploads
```

### ⚠️ Token 有效期

Token 有效期为 7 天，如果测试时提示"登录已过期"，请重新登录获取新 Token。
