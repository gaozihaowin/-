# 营期报名接口完整代码追溯报告

## 一、Mapper 层 - selectCampForEnroll

**文件路径**：`src/main/resources/mapper/CampMapper.xml`（第 159-171 行）

```xml
<select id="selectCampForEnroll" resultType="com.daily.dailychineseculture.entity.Camp">
    SELECT
        camp_id,
        start_time,
        end_time,
        CASE
            WHEN NOW() &lt; start_time THEN 0
            WHEN NOW() > end_time THEN 2
            ELSE 1
        END AS status
    FROM t_camp
    WHERE camp_id = #{campId}
</select>
```

**说明**：当前 SQL 使用 **CASE WHEN 动态计算 status**，而不是直接查询物理字段 `status`。

| 条件 | 计算结果 |
|------|---------|
| NOW() < start_time | status = 0（待开课） |
| NOW() > end_time | status = 2（已结束） |
| 其他情况 | status = 1（进行中） |

## 二、Service 层 - enrollCamp 方法

**文件路径**：`src/main/java/com/daily/dailychineseculture/service/impl/CampServiceImpl.java`（第 207-239 行）

```java
@Override
@Transactional(rollbackFor = Exception.class)
public void enrollCamp(Long userId, Integer campId) {
    if (userId == null) {
        throw new IllegalArgumentException("用户未登录");
    }
    if (campId == null) {
        throw new IllegalArgumentException("campId 不能为空");
    }

    Camp camp = campMapper.selectCampForEnroll(campId);
    if (camp == null) {
        throw new IllegalArgumentException("营期不存在");
    }
    if (camp.getStatus() == null || camp.getStatus() == 2) {  // ← 第 221 行
        throw new IllegalArgumentException("当前营期不可报名");  // ← 第 222 行
    }

    Integer count = campEnrollmentMapper.countByUserIdAndCampId(userId, campId);
    if (count != null && count > 0) {
        throw new IllegalArgumentException("您已报名过该营期，请勿重复操作");
    }

    try {
        int inserted = campEnrollmentMapper.insertEnrollment(userId, campId);
        if (inserted <= 0) {
            throw new RuntimeException("报名失败");
        }
    } catch (DuplicateKeyException e) {
        throw new IllegalArgumentException("您已报名过该营期，请勿重复操作");
    }

    int updated = campMapper.incrementEnrollCount(campId);
    if (updated <= 0) {
        throw new RuntimeException("更新报名人数失败");
    }
}
```

**抛出"当前营期不可报名"的位置**：
- **文件**：CampServiceImpl.java
- **行号**：第 221-222 行
- **条件**：`if (camp.getStatus() == null || camp.getStatus() == 2)`
- **含义**：
  - `camp.getStatus() == null` → status 为空（理论上使用 CASE WHEN 不应发生）
  - `camp.getStatus() == 2` → 营期已结束

## 三、Controller 层 - enrollCamp 接口

**文件路径**：`src/main/java/com/daily/dailychineseculture/controller/CampController.java`（第 103-121 行）

```java
@PostMapping("/enroll")
public ResponseResult<Void> enrollCamp(@RequestBody CampEnrollDTO dto, HttpServletRequest request) {
    try {
        if (dto == null || dto.getCampId() == null) {
            return ResponseResult.error(400, "campId 不能为空");
        }
        // 这里从请求上下文获取 userId；如你的项目使用独立 Token 工具，可替换为 Token 解析逻辑
        Long userId = (Long) request.getAttribute("userId");
        if (userId == null) {
            return ResponseResult.error(401, "未登录或登录已过期");
        }
        campService.enrollCamp(userId, dto.getCampId());
        return ResponseResult.success("报名成功", null);
    } catch (IllegalArgumentException e) {
        return ResponseResult.error(400, e.getMessage());
    } catch (Exception e) {
        return ResponseResult.error("报名失败：" + e.getMessage());
    }
}
```

## 六、可能的问题原因

如果使用 **CASE WHEN** 后仍然报错，可能的原因：

| 场景 | 可能原因 | 解决方案 |
|------|---------|---------|
| status == null | SQL 未返回 status 列 | 确认 XML 中有 `END AS status` |
| status == 2 | 营期确实已结束（end_time < NOW()） | 这是正常逻辑，无法报名 |
| 数据库物理 status 字段 | 如果物理 status=2 但时间上未结束 | 需要确认业务逻辑是否以物理字段为准 |
| 时区问题 | NOW() 与数据库时区不一致 | 检查 MySQL 时区配置 |

## 七、关键确认事项

1. **当前 SQL 使用的是 CASE WHEN 动态计算**，不是物理字段
2. **理论上 status 不应为 null**（CASE WHEN 会返回 0/1/2 之一）
