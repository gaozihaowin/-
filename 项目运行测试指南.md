# 项目运行和测试指南

## 🚀 项目启动步骤

### 1. 设置环境变量（每次启动前执行）
```powershell
$env:JAVA_HOME = "C:\Program Files\Microsoft\jdk-21.0.10.7-hotspot"
$env:Path += ";C:\Program Files\Microsoft\jdk-21.0.10.7-hotspot\bin"
```

### 2. 启动项目
```bash
./mvnw spring-boot:run
```

### 3. 确认启动成功
看到以下信息表示启动成功：
```
Tomcat started on port 8080 (http) with context path '/'
Started DailyChineseCultureApplication in X.XXX seconds
```

## 🧪 API测试方法

### 方法一：使用浏览器测试
访问以下URL进行简单测试：
- http://localhost:8080/api/auth/login (POST请求，需要发送JSON数据)

### 方法二：使用Postman/Insomnia
1. 创建POST请求
2. URL: http://localhost:8080/api/auth/login
3. Headers: Content-Type: application/json
4. Body (JSON):
```json
{
  "username": "admin",
  "password": "123"
}
```

### 方法三：使用命令行curl（推荐）

#### 管理员登录测试
```bash
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"admin\",\"password\":\"123\"}"
```

#### 新用户注册测试
```bash
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"testuser123\",\"password\":\"123456\"}"
```

#### 参数验证测试
```bash
# 空用户名测试
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"\",\"password\":\"123\"}"

# 空密码测试
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"testuser\",\"password\":\"\"}"
```

## 🛠️ 常见问题解决

### 1. 端口被占用
```powershell
# 查找占用8080端口的进程
Get-NetTCPConnection -LocalPort 8080

# 强制结束进程
Get-NetTCPConnection -LocalPort 8080 | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }
```

### 2. Java环境问题
确保已设置JAVA_HOME环境变量：
```powershell
echo $env:JAVA_HOME
# 应该输出: C:\Program Files\Microsoft\jdk-21.0.10.7-hotspot
```

### 3. 项目编译错误
```bash
# 清理并重新编译
./mvnw clean compile

# 运行测试
./mvnw test
```
