## 项目技术栈分析
- **Java版本**: Java 21

- **ORM框架**: MyBatis

## 环境变量配置

### JAVA_HOME 设置
1. 安装JDK后设置系统环境变量
2. 变量名: `JAVA_HOME`
3. 变量值: JDK安装路径 (例如: `C:\Program Files\Java\jdk-21`)

### Path 添加
在系统PATH变量中添加:
- `%JAVA_HOME%\bin`

## 项目启动步骤

### 1. 克隆项目
```bash
git clone <项目地址>
cd daily-chinese-studies
```

### 2. 导入IDE
- IntelliJ IDEA: File → Open → 选择项目目录
- Eclipse: File → Import → Existing Maven Projects

### 3. 数据库配置
修改 `src/main/resources/application.yml`:
```yaml
spring:
  datasource:
    url: jdbc:mysql://你的数据库IP:3306/camp_system?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true&useSSL=false
    username: 你的用户名
    password: 你的密码
```

### 4. 初始化数据库
执行 `数据库代码.txt` 中的SQL脚本创建数据库表

### 5. 运行项目
```bash
# 使用Maven Wrapper
./mvnw spring-boot:run

# 或者在IDE中直接运行DailyChineseCultureApplication类
```

## 常见问题解决

### 1. JAVA_HOME错误
确保JAVA_HOME环境变量正确设置，并重启IDE

### 2. Lombok不生效
- IntelliJ IDEA: 安装Lombok插件并启用注解处理
- Eclipse: 安装Lombok并运行 lombok.jar

### 3. 数据库连接失败
- 检查MySQL服务是否启动
- 确认数据库地址、用户名、密码正确
- 检查防火墙设置
